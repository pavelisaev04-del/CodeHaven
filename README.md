# Учёт почтовых расходов

Локальное веб-приложение для автоматизации учёта расходов на отправку корреспонденции через Почту России.

Загружаете скан или PDF-чека → программа распознаёт данные через OCR → вы проверяете и сохраняете → файл автоматически переименовывается и перемещается на сервер, строка добавляется в общий Excel-файл.

---

## Возможности

- Загрузка чеков в форматах PDF, PNG, JPG, JPEG, TIFF, BMP, GIF
- Автоматическое распознавание через Tesseract OCR:
  - дата отправки (форматы ДД.ММ.ГГГГ и ДД.ММ.ГГ)
  - трек-номер РПО (14 цифр, международный формат)
  - отправитель и получатель
  - итоговая сумма
- Форма проверки с ручной корректировкой перед сохранением
- Автоматическое переименование файла: `ГГГГММДД_Отправка почты Получатель.pdf`
- Перемещение файла в папку проекта на сервере: `R:\[Проект]\[N. Расходы]\`
- Запись строки в основной Excel-файл `20190821_Таблица расходов по проектам.xlsx` с продолжением нумерации
- Локальная база данных SQLite
- Экспорт в Excel с полным форматированием
- Отчёты по месяцам, кварталам, годам и проектам
- Поиск по любым полям
- Доступ из локальной сети

---

## Требования

- Windows 10/11 или Windows Server
- Python 3.10+
- Tesseract OCR 5.x

---

## Установка

### 1. Tesseract OCR

Скачать установщик: https://github.com/UB-Mannheim/tesseract/wiki

При установке обязательно добавить языковые пакеты **Russian** и **English**.

Путь по умолчанию: `C:\Program Files\Tesseract-OCR\tesseract.exe`

### 2. Библиотеки Python

```
pip install flask pymupdf pytesseract Pillow openpyxl
```

### 3. Файлы приложения

Скопировать папку `mail_expenses` в `C:\mail_expenses\`

Структура:
```
C:\mail_expenses\
    app.py
    config.py
    database.py
    ocr_handler.py
    file_handler.py
    excel_handler.py
    requirements.txt
    templates\
    static\
```

## Настройка (config.py)

Открыть `C:\mail_expenses\config.py` и указать пути под свою систему:

```python
# Корневая папка на сервере (диск R:)
STORAGE_BASE = 'R:\\'

# Подпапка внутри каждого проекта (ищется автоматически по слову «расходы»)
STORAGE_EXPENSES_SUBFOLDER = '10. Расходы'

# Путь к основному Excel-файлу на сервере
EXCEL_MASTER_FILE = r'R:\00_РАБОЧАЯ\20190821_Таблица расходов по проектам.xlsx'

# Название листа в Excel-файле
EXCEL_MASTER_SHEET = 'Общие расходы'

# Путь к Tesseract
TESSERACT_PATH = r'C:\Program Files\Tesseract-OCR\tesseract.exe'

# Порт сервера
PORT = 5000
```

---

## Запуск

```
cd C:\mail_expenses
python app.py
```

Открыть в браузере: **http://localhost:5000**

Для доступа из локальной сети использовать IP-адрес компьютера: `http://192.168.x.x:5000`

Чтобы приложение работало постоянно в фоне — настройте его как службу Windows через NSSM или Task Scheduler.

## Структура папок на сервере

Папки проектов должны называться точно так же, как в поле «Наименование проекта» в форме:

```
R:\
    149. БЛИНОВСКАЯ\
        10. Расходы\
            20260113_Отправка почты Останин А.О..pdf
    158. ГОНЧАРОВА\
        5. Расходы\
            20260115_Отправка почты Арбитражный суд.pdf
    00_РАБОЧАЯ\
        20190821_Таблица расходов по проектам.xlsx
```

Подпапка «Расходы» ищется автоматически по названию — номер перед словом не важен (`10. Расходы`, `5. Расходы`, `Расходы` — всё работает).

---

## Excel-файл на сервере

Программа дописывает строки в существующий файл `20190821_Таблица расходов по проектам.xlsx`, лист `Общие расходы`.

Структура таблицы:

| А | Б | В | Г | Д | Е | Ж | З |
|---|---|---|---|---|---|---|---|
| № | Дата оплаты | РПО | Назначение расходов | Отправитель | Получатель | Сумма | Проект |

- Нумерация продолжается автоматически от последней строки
- Столбец «Назначение расходов» всегда заполняется значением «отправка почты»
- Если файл открыт другим пользователем — выводится предупреждение, запись в БД сохраняется

## Поддерживаемые форматы чеков Почты России

| Формат РПО | Пример |
|---|---|
| С ведущей цифрой | `1 РПО № 80110619875732` |
| Без ведущей цифры | `РПО № 80515419896756, шт.` |
| С двоеточием | `РПО: 80110619875732` |
| Международный | `RX123456789RU` |

| Формат суммы | Пример |
|---|---|
| С равно | `ИТОГ =133.60` |
| С минусом (расход) | `ИТОГ -138.60` |
| С пробелами (PDF) | `ИТОГ    133.60` |
