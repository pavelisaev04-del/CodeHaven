# CodeHaven
# Юридический AI-ассистент и онлайн-сервис продажи шаблонов

Этот проект представляет собой онлайн-платформу для автоматизации юридических консультаций и продажи готовых шаблонов документов (претензии, заявления, чек-листы). Сервис предназначен для малых и средних предпринимателей, юристов и физических лиц, которым нужна быстрая квалификация обращения, предварительная оценка ситуации и удобный доступ к готовым юридическим документам.  

Проблема, которую решает проект: клиенты часто тратят много времени на поиск правильных шаблонов и первичное оформление документов, а юристы теряют ресурсы на обработку рутинных заявок. Сервис минимизирует ручную работу, ускоряет первичную квалификацию и позволяет получать пассивный доход через продажу готовых документов.

## Features
- Автоматическая первичная квалификация обращений через бота с 5 уточняющими вопросами.  
- Продажа шаблонов документов через сайт и Telegram с мгновенной выдачей после оплаты.  
- Интеграция с онлайн-календарями (Calendly, Tidycal) для автоматической записи на консультации и отправки ссылок Zoom.  
- AI-ассистент для анализа ситуации клиента и генерации уточняющих вопросов для юриста.  
- Модульная архитектура для расширения функций, новых шаблонов и каналов взаимодействия.  

## Installation

1. Клонируйте репозиторий:  
   ```bash
   git clone https://github.com/username/legal-ai-assistant.git
   cd legal-ai-assistant
   python -m venv venv
   
2. Создайте виртуальное окружение (Python 3.11+):
source venv/bin/activate   # Linux / macOS
venv\Scripts\activate      # Windows
3. Установите зависимости:
pip install -r requirements.txt
4. Настройте переменные окружения для Telegram, AI API и Zoom:
export TELEGRAM_TOKEN='ваш_токен'
export AI_API_KEY='ваш_api_ключ'
export ZOOM_API_KEY='ваш_zoom_ключ'
5. Запустите проект:
python main.py
