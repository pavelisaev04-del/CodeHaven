# Учёт почтовых расходов

Локальная веб-программа для автоматизации учёта расходов на почтовые отправления.

Открывается в браузере. Работает на Windows и Windows Server.
Данные хранятся локально в базе данных SQLite.

---

## Что умеет программа

- Загружать фото, скан или PDF-файл почтового чека
- Автоматически распознавать текст и заполнять поля
- Показывать форму проверки — пользователь проверяет и при необходимости исправляет данные
- Сохранять запись в базу данных
- Переименовывать и перемещать файл чека в папку проекта
- Искать и фильтровать записи по датам, РПО, проекту, отправителю, получателю
- Формировать отчёты за месяц, квартал, год, по проектам
- Экспортировать данные в Excel

## Установка на Windows

### Шаг 1. Установить Python

1. Скачайте Python 3.11 или новее: https://python.org/downloads/
2. При установке **обязательно** поставьте галочку **«Add Python to PATH»**
3. Проверьте в командной строке:
   ```
   python --version
   ```

### Шаг 2. Установить зависимости Python

Откройте командную строку в папке программы и выполните:

```bat
pip install -r requirements.txt
```

### Шаг 3. Установить Tesseract OCR

Tesseract нужен для распознавания текста с изображений (JPG, PNG).
Для PDF со встроенным текстом Tesseract не нужен.

1. Скачайте установщик для Windows:
   https://github.com/UB-Mannheim/tesseract/wiki

2. Во время установки выберите дополнительный язык — **Russian (rus)**

3. Путь установки по умолчанию:
   ```
   C:\Program Files\Tesseract-OCR\tesseract.exe
   ```

4. Если вы установили Tesseract в другую папку, откройте файл `config.py`
   и измените строку:
   ```python
   TESSERACT_PATH = r'C:\Program Files\Tesseract-OCR\tesseract.exe'
   ```

### Шаг 4. Запустить программу

**Способ 1 — двойной клик:**
Дважды кликните на файл `run.bat`

**Способ 2 — командная строка:**
```bat
python app.py
```

После запуска откройте браузер и перейдите по адресу:
```
http://localhost:5000
```

## Доступ из локальной сети

Если программа запущена на сервере или другом компьютере,
сотрудники могут открыть её по IP-адресу компьютера:

```
http://192.168.1.100:5000
```

(замените 192.168.1.100 на реальный IP-адрес компьютера)

Узнать IP-адрес: откройте командную строку и введите `ipconfig`

> **Важно:** брандмауэр Windows может блокировать порт 5000.
> Если другие компьютеры не могут подключиться, добавьте разрешение:
> Панель управления → Брандмауэр Windows → Дополнительные параметры →
> Правила для входящих подключений → Создать правило → Порт 5000 → TCP

## Запуск на Windows Server

### Вариант 1 — ручной запуск (простой)

Подключитесь к серверу через RDP и запустите `run.bat` или команду:
```bat
python app.py
```

### Вариант 2 — автозапуск через Планировщик задач

1. Откройте **Планировщик задач** (Task Scheduler)
2. Создайте новую задачу
3. Триггер: **При запуске системы**
4. Действие: **Запуск программы**
   - Программа: `python`
   - Аргументы: `C:\mail_expenses\app.py`
   - Рабочая папка: `C:\mail_expenses`
5. Дополнительно: **Выполнять независимо от входа пользователя**

### Вариант 3 — как Windows-служба (через NSSM)

1. Скачайте NSSM: https://nssm.cc/download
2. В командной строке (от администратора):
   ```bat
   nssm install MailExpenses python "C:\mail_expenses\app.py"
   nssm set MailExpenses AppDirectory C:\mail_expenses
   nssm start MailExpenses
   ```

## Как изменить папку для хранения чеков

Откройте файл `config.py` и найдите строку:

```python
STORAGE_FOLDER = os.path.join(BASE_DIR, 'storage')
```

Замените на нужный путь, например:

```python
# Другой диск
STORAGE_FOLDER = r'D:\Документы\Расходы'

# Сетевая папка
STORAGE_FOLDER = r'\\server\share\Расходы'

# Папка на рабочем столе
STORAGE_FOLDER = r'C:\Users\Иванов\Desktop\Расходы'
```

После изменения перезапустите программу.

---

## Как изменить порт

По умолчанию программа работает на порту **5000**.
Чтобы изменить, откройте `config.py`:

```python
PORT = 8080  # или любой другой свободный порт
```

## Как работает переименование файлов

Загруженный файл чека переименовывается по шаблону:

```
ГГГГММДД_Отправка почты Получатель.расширение
```

Пример:
```
20260122_Отправка почты Арбитражный суд города Москвы.pdf
```

Если файл с таким именем уже существует, добавляется номер:
```
20260122_Отправка почты Арбитражный суд города Москвы_2.pdf
```

Файл перемещается в папку:
```
storage/
└── Название проекта/
    └── 20260122_Отправка почты Арбитражный суд города Москвы.pdf
```

## Возможные проблемы

**Ошибка «python не является внутренней командой»**
→ Python не установлен или не добавлен в PATH. Переустановите Python с галочкой «Add to PATH».

**Tesseract не найден**
→ Проверьте путь в `config.py`. Убедитесь, что Tesseract установлен.

**Порт 5000 занят**
→ Измените PORT в `config.py` на другой, например 5001 или 8080.

**Файл не распознаётся**
→ Убедитесь, что pytesseract и Pillow установлены. Для PDF — PyMuPDF.
→ Все поля можно заполнить вручную даже без OCR.
